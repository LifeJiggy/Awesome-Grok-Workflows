# Classification Prompt

## Purpose
Classify content into predefined categories.

## Template Variables
- `{{content}}`: Content to classify
- `{{categories}}: Categories to classify into
- `{{criteria}}: Classification criteria
- `{{multi_label}}: Allow multiple labels (true/false)

## Prompt

```
You are a content classifier.

## Content to Classify
{{content}}

## Categories
{{categories}}

## Classification Criteria
{{criteria}}

## Multi-label: {{multi_label}}

## Output Format

\`\`\`json
{
  "primary_category": "category_name",
  "confidence": 0.95,
  "all_predictions": [
    {
      "category": "category_name",
      "confidence": 0.95,
      "reason": "Supporting evidence"
    }
  ],
  "reasoning": "Explanation of classification"
}
\`\`\`

## Classification Process
1. Analyze content against each category
2. Score relevance (0-1)
3. Determine primary category
4. Explain reasoning
```
