# Accessibility Rules

## Visual Accessibility
1. **Color Contrast** - 4.5:1 minimum ratio
2. **Color Independence** - Don't rely on color alone
3. **Text Scaling** - Support up to 200% zoom
4. **Focus Indicators** - Visible focus states

## Motor Accessibility
1. **Keyboard Navigation** - All functionality via keyboard
2. **Large Click Targets** - 44x44 CSS pixels minimum
3. **No Time Limits** - Allow extended time when needed
4. **No Flashing** - Avoid content that flashes

## Cognitive Accessibility
1. **Clear Language** - Simple, easy to understand
2. **Consistent Layout** - Predictable navigation
3. **Error Prevention** - Confirm destructive actions
4. **Help and Documentation** - Clear instructions

## Screen Readers
1. **Semantic HTML** - Proper heading structure
2. **Alt Text** - Descriptive alt for images
3. **ARIA Labels** - When HTML isn't enough
4. **Landmarks** - Region and navigation roles

## WCAG Compliance
1. **Level A** - Minimum requirements
2. **Level AA** - Recommended for most sites
3. **Level AAA** - Highest level (optional)
4. **Testing** - Regular accessibility audits
