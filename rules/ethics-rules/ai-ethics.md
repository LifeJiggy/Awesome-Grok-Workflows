# Ethics Rules

## AI Ethics
1. **Transparency** - users should know they're interacting with AI
2. **No deception** - don't pretend to be human
3. **Bias awareness** - acknowledge and mitigate biases
4. **Privacy protection** - don't store personal data without consent

## Content Safety
1. **No harmful content** - violence, self-harm, illegal acts
2. **No misinformation** - provide accurate information
3. **No hate speech** - respect all users
4. **No harassment** - respectful interactions only

## Accessibility
1. **Inclusive design** - works for all users
2. **Multiple formats** - text, audio, visual
3. **Compliance** - WCAG 2.1 AA standards
4. **Accommodations** - support different needs

## Data Ethics
1. **Data minimization** - collect only what's needed
2. **Purpose limitation** - use data only as stated
3. **User control** - users can access/delete their data
4. **Anonymization** - remove personally identifiable info

## Environmental Impact
1. **Energy efficiency** - optimize resource usage
2. **Carbon awareness** - consider environmental cost
3. **Sustainable practices** - green hosting, efficient code

## Societal Impact
1. **Consider consequences** - how might this be misused?
2. **Beneficial use** - aim to help, not harm
3. **Fair access** - don't create digital divides
4. **Economic impact** - consider job displacement

## Accountability
1. **Clear ownership** - responsible parties
2. **Audit trails** - track decisions
3. **Appeal process** - human review available
4. **Continuous improvement** - learn from mistakes
