# Incident Response Rules

## Preparation
1. **Response Plan** - Documented procedures
2. **Team Training** - Regular drills
3. **Tooling** - Ready-to-use tools
4. **Communication** - Contact information

## Identification
1. **Monitoring** - Detect incidents early
2. **Alerting** - Notify quickly
3. **Triage** - Assess severity
4. **Escalation** - Proper channels

## Containment
1. **Short-term** - Immediate containment
2. **Long-term** - Sustained containment
3. **Evidence Preservation** - Collect evidence
4. **Scope** - Determine full extent

## Eradication
1. **Root Cause Analysis** - Find the cause
2. **Remove Threat** - Delete malware, close holes
3. **Verify** - Confirm eradication
4. **Validate** - System is clean

## Recovery
1. **Restore from Backup** - Clean backups
2. **Validate** - Test before going live
3. **Monitor** - Enhanced monitoring
4. **Confirm** - Back to normal

## Post-Incident
1. **Documentation** - Complete incident report
2. **Lessons Learned** - What went wrong
3. **Process Improvement** - Update procedures
4. **Share Knowledge** - Team awareness
