# Content Creation Rules

## General Principles

### Quality Standards
- **Accuracy**: Verify all facts before publishing
- **Clarity**: Clear, concise language
- **Completeness**: Cover topic thoroughly
- **Consistency**: Maintain voice and style
- **Value**: Provide unique insights

### Voice and Tone
| Context | Tone | Example |
|---------|------|---------|
| Technical | Professional, precise | "The algorithm achieves O(n log n) complexity" |
| Blog | Engaging, accessible | "Here's the secret sauce behind..." |

## SEO Rules

### On-Page SEO
```yaml
optimization:
  title:
    length: 50-60 characters
    include: primary keyword
  
  meta_description:
    length: 150-160 characters
    include: keyword, CTA
  
  headings:
    h1: 1 per page (main topic)
    h2: 3-5 per page (major sections)
    h3: As needed (subsections)
  
  keywords:
    primary: 1 per article
    secondary: 3-5 per article
    density: 1-2%
```

### Content Structure
- Use short paragraphs (2-3 sentences)
- Include bullet points and lists
- Add relevant internal links
- Optimize images (alt text, compression)
- Use descriptive URLs

## Visual Content

### Image Guidelines
| Type | Format | Max Size | Purpose |
|------|--------|----------|---------|
| Photos | WebP/JPEG | 100KB | Context |
| Diagrams | SVG | 50KB | Explanation |
| Screenshots | WebP | 200KB | Tutorial |
| Icons | SVG | 10KB | Visual aid |

### Design Principles
1. **Contrast**: Ensure readability
2. **White space**: Don't overcrowd
3. **Alignment**: Grid-based layout
4. **Hierarchy**: Clear visual order
5. **Consistency**: Same style throughout

## Accessibility

### Text Accessibility
- **Contrast ratio**: 4.5:1 minimum
- **Font size**: 16px base minimum
- **Line height**: 1.5 minimum
- **Paragraph width**: 60-80 characters
- **Plain language**: Avoid jargon

### Format Accessibility
- **Alt text** for all images
- **Captions** for videos
- **Transcripts** for audio
- **Semantic HTML** structure
- **Descriptive links** (not "click here")

## Legal Compliance

### Copyright
- Use licensed or original images
- Credit sources appropriately
- Respect fair use guidelines
- Don't plagiarize
- Check terms of service

### Disclosure
- Disclose sponsored content
- Disclose affiliate links
- Be transparent about sources
- Note AI-generated content
- Include corrections/updates

## Metrics and Analytics

### Track These Metrics
| Metric | What It Measures | Target |
|--------|------------------|--------|
| Page views | Visibility | Growth MoM |
| Time on page | Engagement | >2 minutes |
| Bounce rate | Relevance | <60% |
| Social shares | Virality | >5% share rate |
| Comments | Discussion | Active discussion |

### Continuous Improvement
1. Review analytics weekly
2. A/B test headlines
3. Update underperforming content
4. Repurpose popular topics
5. Remove outdated content
