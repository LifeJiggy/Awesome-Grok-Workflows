# Cloud Cost Optimization Rules

## Right-Sizing
1. **Analyze Usage** - Match resources to needs
2. **Start Small** - Begin with smaller instances
3. **Scale Up** - Increase when needed
4. **Monitor** - Track usage patterns

## Reserved Capacity
1. **Commit to Use** - Reserved instances
2. **Savings Plans** - Flexible commitment
3. **Spot Instances** - For fault-tolerant workloads
4. **Negotiate** - Enterprise agreements

## Storage Optimization
1. **Lifecycle Policies** - Auto-tier data
2. **Compression** - Reduce storage costs
3. **Deduplication** - Eliminate duplicates
4. **Right Storage Class** - Match access patterns

## Network Efficiency
1. **Minimize Transfer** - Reduce data movement
2. **CDN Usage** - Cache at edge
3. **Compression** - Reduce bandwidth
4. **Direct Connect** - For high-volume traffic

## FinOps
1. **Cost Visibility** - Track all costs
2. **Chargeback** - Show costs to teams
3. **Budget Alerts** - Notify before overspend
4. **Regular Reviews** - Quarterly optimization
